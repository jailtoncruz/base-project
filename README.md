# Base project (Tom Cruz)

This project is a base for my projects, already installed and configured:

- Prisma
- Authentication (JWT)
- Tailwind
- Radix
- Docker
- Short Unique ID
- SendGrid
- Swagger
- Toast (react-toast)
- React Loading Skeleton

## Installation

```sh
npm install
```

## Usage

```sh
npm run start
```

## Tests

```sh
npm run tests
```
